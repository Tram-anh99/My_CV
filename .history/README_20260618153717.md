
# 🌐 Trang Web Cá Nhân: CV, Portfolio & Blog Động

Đây là dự án trang web cá nhân của tôi, kết hợp giữa hồ sơ năng lực (CV), nơi lưu trữ các dự án và một hệ thống Blog đăng bài viết hằng ngày/quý.

## 🛠️ Công nghệ sử dụng

- **Backend:** Python, FastAPI
- **Database:** SQLite (Nhúng nhỏ gọn)
- **Frontend:** HTML5, CSS3, Jinja2 Templates

## 🚀 Tính năng hiện tại

- [X] Hiển thị thông tin cá nhân (CV) cố định.
- [X] Danh sách các đường dẫn dự án cá nhân.
- [X] Blog động: Tự động tải bài viết từ cơ sở dữ liệu SQLite lên giao diện.
